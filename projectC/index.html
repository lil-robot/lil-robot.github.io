Javascript JavaScript é uma linguagem de script orientada a objetos
e plataforma cruzada usada para tornar as páginas da Web interativas		 1 ano e 2 meses intermediário/avançado
HTML HTML (Linguagem de Marcação de HiperTexto) é o bloco
de construção mais básico da web. Define o significado
e a estrutura do conteúdo da web		 4 meses intermediário
CSS CSS é chamado de linguagem Cascading Style Sheet e é usado
para estilizar elementos escritos em uma linguagem de marcação como HTML.		 2 meses iniciante
Dart Dart é uma linguagem de programação multiparadigma e multiplataforma criada pela Google em 2011 1 mês iniciante
